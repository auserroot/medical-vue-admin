# 项目简介
## 该项目使用vue-cli搭建
### 其中使用了RAP、 ELEMENT-UI、Echarts 、axios、FileSave、amap 等插件/组件 实现页面构建和布局
## 基本功能简介
### 登录：有基本逻辑验证（由于请求接口是rap故无法拥有真实体验）
### 首页：数据面板 主要由 Echarts 构建用于图形化数据查看
### 各个管理面板 基于Element-ui 用于查看和管理数据 可实现数据 增删改查 功能
